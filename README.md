# Microsite — Ski Areál Praha (beginner-friendly)

Co obsahuje:
- Responzivní one-page postavená na Bootstrapu 5.
- Velký hero (3/4 výšky okna) s fallback obrázkem.
- Sekce: O areálu, Co nabízíme (grid karet), Rezervace (form + newsletter), Video, Mapa (Petřín), Kontakt, Footer.
- Jednoduché klientské skripty pro validaci a UX.

Jak nasadit lokálně:
1. Vytvoř složku `microsite/`.
2. Ulož sem soubory: `index.html`, `styles.css`, `script.js`, `README.md`.
3. Vytvoř podsložku `microsite/images/` a vlož sem vlastní obrázky:
   - `images/hero.jpg` (doporučeno: šířka >= 2000px)
   - `images/service1.jpg`, `images/service2.jpg`, `images/service3.jpg` (nepovinné — stránka používá Unsplash fallback)
4. Otevři `index.html` ve VS Code a spusť Live Server nebo otevři přímo v prohlížeči.

Poznámky:
- Tohle je klient‑side demo. Pro skutečné odesílání e‑mailů nebo ukládání rezervací je třeba backend (mohu přidat ukázkový POST endpoint).
- Kód je úmyslně jednoduchý a čitelný pro začátečníky — ideální jako školní práce.
